# HertzOS
A simple web smartwatch OS written in pure HTML, CSS, and JS. The main theme is to create the most minimalistic and modern UI with slight glass morphism.
###### Good to know:
- No AI has been used for this project.
- Later on I will be trying to port this project into an actual OS using Python as a backend with the help of the Linux kernel.
- 99% of the assets are made by me.
